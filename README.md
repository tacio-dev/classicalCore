# Classical Core
**Classical Core** é um projeto front-end conceitual que apresenta três dos maiores compositores da música erudita ocidental: Johann Sebastian Bach, Wolfgang Amadeus Mozart e Ludwig van Beethoven.
A proposta do projeto é unir **desenvolvimento web, identidade visual e storytelling**, representando cada compositor de acordo com seu período histórico e sua importância na construção da música clássica.

---

## Objetivo
* Demonstrar habilidades em desenvolvimento front-end
* Aplicar uma identidade visual temática e consistente
* Praticar organização e estruturação de código
* Criar uma experiência que una arte, cultura e tecnologia

---

## Tecnologias utilizadas
* HTML5
* CSS3
* JavaScript

---

## Funcionalidades
* Navegação entre os compositores
* Seções informativas com contexto histórico
* Layout temático inspirado nos períodos musicais
* Design responsivo para diferentes telas

---

## Adaptação do projeto
Este projeto foi originalmente desenvolvido como parte de um curso da Digital Innovation One, cuja proposta inicial era um site sobre o Multiverso Homem-Aranha.

### Modificações realizadas por mim:
* Alteração completa do tema (de super-herói para música clássica)
* Criação de uma nova identidade visual (cores, tipografia e estilo)
* Reestruturação de conteúdo e narrativa
* Customização do CSS para melhor apresentação e consistência visual
